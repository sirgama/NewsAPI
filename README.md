# SirGama Street Journal
<img src="./app/static/Screenshot_select-area_20220503011323.png">


## Description
A web application that uses NEWSAPI to fetch news articles from various sources and in different categories as well. The app is built on Flask framework.

## Live preview
You can view the site at: https://sirgama-sj.herokuapp.com/

## Setup Instructions and Installation

- Clone this repository to a location in your file system. `git clone https://github.com/sirgama/NewsAPI.git`
- Open terminal command line then navigate to the root folder of the application. `cd NewsAPI
- Run `./run.py`
